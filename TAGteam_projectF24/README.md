# TAGteam-projectF24

This folder contains the documentation for the TAGteam project from Fall 2024. 

## Folder Structure

/main.sh
Contains the main code reuired to get the positions of TAGteam motifs

/vis.Rmd
Contains the main code to get visualizations (see below for example)

/gene_name
Contains sample gene_name_protein.fasta file. For each gene of interest, must download the protein FASTA file in ordered to run the code
Contains table.csv, a csv that sets up the base pair regions that are to be search for TAGteams for each species
